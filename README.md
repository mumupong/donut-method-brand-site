# Donut Method brand site

Next.js App Router で作った、ドーナツメソッドのブランドサイト一式です。

## 使い方

```bash
npm install
npm run dev
```

## 最初に差し替える場所

`lib/site-data.ts` の以下を、ご自身の情報に置き換えてください。

- `sessionConfig.priceLabel` 料金
- `sessionConfig.bookingUrl` 予約URL
- `sessionConfig.contactEmail` お問い合わせ先
- `sessionConfig.cancellationText` キャンセルポリシー
- `resources` note / YouTube の記事・動画データ
